### プロフィールページ

ひとまず プロフィールページ は、 tailwind css によって出来てるんだけど、雑にパッケージをアップデートしたら、 postcss が分離してるだとか、 config に適用し直してね！ってエラーが出てきたり ... 2年も放置すれば壊れちゃうんですね ...

サイトとしては以前よりも面白いものが減っています。以前は navbar があったんだけど、そもそもこんなものいらないよねって消し飛ばして、1ページですべてが見えるようになった

技術的なことはそんな感じなんだけど、スタイルとかを大きくいじったりはしてなくて ... 引き続き tailwind 、なにかいい手段があるならそれに乗り換えてもいい気がしてくるけど、どういうのがいいんだろうね？

### ブログ

ブログを建てようと思ったときに、使い勝手としては [TexTransTool の ブログ](https://ttt.rs64.net/blog) とあんまり変わらない、 MarkDown をいい感じにレンダリングできて、 youtube とか動画を載せれれば何でも良かったから ...

hugo を、テーマを [hugo-paper](https://github.com/nanxiaobei/hugo-paper) にして、コンフィグを少しいじって今の形に、シンプルにブログだけがあるサイトであってほしくて、他に変なものがあまりついてなさそうな範囲だったのでこのテーマを選んでいます。

とりあえずこれも、クラウドフレアになんとかしてもらってデプロイしよーねー！
